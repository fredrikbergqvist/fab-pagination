fab-pagination.js

An accessible pagination plugin for jQuery

Installation

Download the script and place it in your projects script folder.

Requirements

You must use the jQuery library.

Tested with:

  • jQuery 1.10.2

Restrictions

The plugin currently only supports paginating one table/ul/etc at a time. Should you need to paginate two tables on the same page, this plugin is not for you (yet)

Usage

To use the fab-pagination plugin simply include the plugin's JavaScript file in your code. Make sure the script tag is placed below jQuery.

<script src="path/to/scripts/fab-pagination.jquery.min.js"></script>

Then attach fab-pagination to the element you want to paginate.

$("table").fabPagination();

If you want to make any changes in the plugins behaviour you will need to change the settings

Settings

These are the default settings.

{
itemsPerPage : 10,
rowType: "tbody tr",
paginationContainer : ".fab-pagination-container"
}

itemsPerPage

The number of items you want to be displayed per page.

rowType

By default fab-paginator assumes you want to paginate a table, however if you want to page a list or something else you will have to set the row element here (li for list etc).

paginationContainer

If you want to place the pagination list somewhere else you can specify a container here.

Styling

Css-classes

The default pagination module looks like this:

<ul role="navigation" aria-labelledby="fabPaginationLabel" class="fab-pagination-container"> <li> <a href="#" class="fab-pagination-link-previous" rel="prev" data-from="0" data-to="0"> <span class="fab-pagination-audible">Previous </span><span><</span> </a> </li> <li> <a href="#" class="fab-pagination-link selected" rel="Page " data-from="0" data-to="10"> <span>1</span> </a></li> <li><a href="#" class="fab-pagination-link" rel="Page " data-from="10" data-to="20"> <span>2</span></a> </li> <li><a href="#" class="fab-pagination-link-next" rel="next" data-from="0" data-to="0"> <span class="fab-pagination-audible">Next </span><span>></span></a></li> </ul>
